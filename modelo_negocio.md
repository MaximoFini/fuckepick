# Análisis del Modelo de Negocio — E-Pick (Argentina)

## 2. Propuesta de Valor

| Actor            | Valor ofrecido                                                                 |
|------------------|------------------------------------------------------------------------------|
| E-commerce       | Reducción de costos logísticos (hasta 30–50% vs envío a domicilio)          |
| Usuario final    | Mayor flexibilidad (retiro cuando quiera, menor tasa de fallas)             |
| Comercios físicos| Generación de tráfico adicional + ingresos por servicio                     |

---

## 3. Fuentes de Ingresos

### 3.1 Ingresos principales

| Fuente                         | Descripción                                                                 |
|--------------------------------|-----------------------------------------------------------------------------|
| Fee por envío                  | Tarifa por paquete gestionado (principal revenue stream)                    |
| Fee por almacenamiento         | Cargo por permanencia en punto de retiro                                   |
| Servicios B2B                  | Integraciones logísticas, fulfillment parcial                               |
| Servicios premium              | Envíos prioritarios o same-day (según zona)                                |

### 3.2 Estructura de pricing (estimada Argentina)

| Tipo de envío         | Precio estimado ARS (2025–2026) |
|-----------------------|---------------------------------|
| Última milla estándar | $2.500 – $4.500                 |
| Pick-up point         | $1.500 – $2.800                 |

**Insight clave:**  
El modelo pick-up reduce costos entre **30% y 50%** frente a delivery tradicional.

---

## 4. Estructura de Costos

### 4.1 Costos variables (principales)

| Categoría                | Descripción                                                     |
|--------------------------|-----------------------------------------------------------------|
| Transporte               | Pago a operadores logísticos (fleteros, carriers)              |
| Operación puntos         | Comisión a comercios adheridos                                 |
| Tecnología               | Infraestructura cloud, tracking, integraciones API             |
| Atención al cliente      | Soporte operativo                                              |

### 4.2 Costos fijos

| Categoría         | Descripción                              |
|-------------------|------------------------------------------|
| Equipo            | IT, operaciones, ventas                  |
| Plataforma        | Desarrollo y mantenimiento               |
| Administración    | Legal, contable                          |

## 5. Red Logística y Partners

### 5.1 Componentes de la red

| Elemento            | Rol                                                            |
|---------------------|----------------------------------------------------------------|
| Pick-up points      | Comercios físicos (kioscos, tiendas, etc.)                     |
| Carriers            | Empresas de transporte tercerizadas                            |
| E-commerce partners | Tiendas online integradas                                     |
| Plataforma IT       | Sistema de gestión, tracking y optimización                    |

### 5.2 Funcionamiento simplificado

1. E-commerce genera envío  
2. E-Pick asigna ruta y punto de retiro  
3. Carrier transporta paquete  
4. Usuario retira en comercio  }
