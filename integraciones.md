# E-Pick: Integraciones Activas

## Plataformas E-commerce Integradas

### 1. Tiendanube

**Status:** ✅ Integración activa oficial  
**Tipo:** App nativa en marketplace Tiendanube  
**Disponibilidad:** Documentada desde abril 2025  
**Regiones:** Argentina, México, Chile  

**Funcionalidades:**
- Generación automática de envíos al recibir pedidos
- Consulta de tarifas en tiempo real
- Impresión de etiquetas desde panel
- Actualización automática de estado de envíos

**Fuentes:**
- [Tiendanube - FAQ E-Pick](https://ayuda.tiendanube.com/es_ES/e-pick/preguntas-frecuentes-sobre-e-pick)
- [Tiendanube - Integración E-Pick](https://ayuda.tiendanube.com/es_ES/e-pick/como-integrar-e-pick-con-mi-tienda)

---

### 2. Empretienda

**Status:** ✅ Integración activa  
**Tipo:** Integración directa desde panel  
**Última actualización documentada:** 2025  

**Funcionalidades:**
- Configuración de método de envío desde panel Empretienda
- Sincronización automática de pedidos
- Opciones de envío gratis o con recargos
- Definición de dirección de retiro

**Fuentes:**
- [Empretienda - Tutorial E-Pick 2025 (PDF)](https://recursos.empretienda.com/tutorial-epickv2.pdf)

---

### 3. WooCommerce

**Status:** ✅ Plugin disponible  
**Tipo:** Plugin desarrollado por terceros  
**Compatibilidad:** WooCommerce 3.x - 5.x  

**Funcionalidades:**
- Cotización y creación de pedidos en tiempo real
- Integración mediante clave API
- Seguimiento de pedidos
- Soporte premium 24/7

**Fuentes:**
- [Pasarelas de Pagos - Plugin WooCommerce E-Pick](https://www.pasarelasdepagos.com/shop/ecommerce-argentina/woocommerce-argentina/e-pick-wordpress-woocommerce-argentina-metodo-de-envio-delivery/)
- [Wanderlust Codes - WooCommerce E-Pick](https://shop.wanderlust-webdesign.com/shop/woocommerce-e-pick/)

---

### 4. VTEX

**Status:** ⚠️ Integración vía API personalizada  
**Tipo:** Requiere desarrollo custom  

**Notas:**
- No existe app oficial en marketplace VTEX
- Integración mediante API REST de E-Pick
- Requiere coordinación técnica entre equipos

## API Pública

**Status:** ✅ Disponible  
**Tipo:** REST API  

**Uso:**
- Integración con plataformas personalizadas
- Desarrollo de conectores custom
- Automatización de procesos logísticos

**Regiones con acceso a API:**
- Argentina: [e-pick.com.ar](https://www.e-pick.com.ar/home)
- Chile: [e-pick.cl/integration](https://www.e-pick.cl/integration/)
